# Node.js global modules
Following are node.js global modules
# Array
This is a simply Javascript `Array`. Following are few example.
```
const array1 = new Array();

console.log(Array.isArray());

const listOfChars = Array.from('hello');
const singleItemArray = Array.of(7);
const multipleItemArray = Array.of(1,3,5);

const emptyArray = Array(7);

```

Following are few instance function.
- Array.prototype.concat()
- Array.prototype.copyWithin()
- Array.prototype.entries()
- Array.prototype.every()
